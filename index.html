DeTour

My capstone project for General Assembly's Web Development Immersive, DeTour, came about because I wanted to build a website that involved an interest of mine, traveling. I was looking to join a tour through Southeast Asia, but there was no one central hub where I could easily compare all the options. DeTour solves that problem. It lists tours from the major tour companies offering group travel for 18-35/40 year-olds in a side-by-side comparison. Users can choose to register for an account if they want to save tours to a wishlist or to share their experiences on tours they've taken.

Tech stack: React, Redux, Sass, Bulma, MongoDB, Express, Node.js

Hangry

Hangry is a platform for users to create and save recipes. It was a group project of 4 members during General Assembly's Web Development Immersive to learn Python and Flask. I worked mostly on the frontend using Jinja2 to create templates displaying data gathered from our SQLite database. During development, to test my templates, I touched the backend to create and test routes and methods to render a template or redirect to a different method. I also was in charge of cleaning up our code to ensure readability, style guidance, and best practices.

Wayfarer
